# 单片机的介绍

**单片机**顾名思义，**单片微型计算机**，它把所有功能压缩到唯一一块芯片里，这里的 “单” 就是单一块、单个的意思，**而不是单个CPU**的意思

从硬件结构来看，单片机整体由四大核心部分构成：CPU内核、片内存储器、片上外设以及时钟、复位、电源等基础电路。

单纯将各个模块分开讲太枯燥了，我就直接讲单片机是怎么运行的吧。

## CPU内核

如何理解CPU内核呢？我们可以从单片机是怎么运行的开始理解

### 内核系列

当前主流单片机大多使用 ARM Cortex-M 系列内核。

STM32F103C8T6	Cortex-M3
STM32F407VGT6	Cortex-M4
STM32F407VET6	Cortex-M4
MSPM0G3507	Cortex-M0+

### 你不需要深究内核细节

作为应用开发者，你几乎不需要直接和 CPU 内核打交道。C 编译器帮你把代码翻译成指令，调试器帮你单步跟踪，HAL 库帮你操作外设。你只需要知道：

- **主频**（如 72MHz、168MHz）：决定了单片机一秒钟能做多少事。越高越快，但功耗越大。
- **架构版本**（ARMv6-M、ARMv7-M）：决定了支持哪些指令和功能。M0/M0+ 是 ARMv6-M，M3/M4/M7 是 ARMv7-M。选型时看一眼就行，平常不用管。

真正占用你开发时间的是外设（GPIO、串口、ADC、定时器……），不是内核本身。好，内核讲完了——接下来看单片机怎么"记住"东西。
