# 引脚电气数据

## 工作电压

单片机工作电压是 3.3V。引脚的高电平是 3.3V，低电平是 0V。

| | 低电平 (LOW) | 高电平 (HIGH) |
|------|-------------|--------------|
| 输出 | ~0V | ~3.3V |
| 输入 | < 0.8V | > 2.0V |

输入电压在 0.8V~2.0V 之间时，芯片可能读成 0 也可能读成 1，结果不可靠。悬空引脚容易落在这个区间。

## 电流限制

GPIO 引脚用来传信号，不是供动力。单引脚最大输出电流约 20mA。

| 可以直接接 | 必须通过驱动电路 |
|-----------|----------------|
| LED（串 220Ω 限流电阻） | 电机 |
| 按钮、信号线 | 继电器 |
| 三极管基极 | 大功率 LED |
