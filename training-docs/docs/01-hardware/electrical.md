# 引脚电气数据 — 能接什么，不能接什么

## 最重要的一条规则

::: danger 工作电压：3.3V
STM32 和 MSPM0 系列单片机的工作电压是 **3.3V**。

**直接给普通引脚接 5V 信号 = 烧毁芯片！**

这不是夸张。5V 高于芯片内部晶体管的耐压极限，轻则烧坏一个引脚，重则整颗芯片报废，而且不可修复。
:::

哪怕你在网上看到有人说"我接过 5V 没事啊"——那是运气好，或者他的引脚恰好是 5V 耐受引脚（见下文）。不要赌概率。

## 电压逻辑电平

单片机的世界只有 0 和 1，但物理上它们对应的是电压：

| 方向 | 逻辑 0 (LOW) | 逻辑 1 (HIGH) | 不确定区 |
|------|-------------|--------------|---------|
| 输出 | 约 0V | 约 3.3V | — |
| 输入 | 低于 0.8V | 高于 2.0V | 0.8V ~ 2.0V 之间不可靠 |

::: warning 不确定区域
如果输入电压在 0.8V ~ 2.0V 之间，单片机可能读成 0，也可能读成 1，甚至同一微秒内反复跳变。这就是为什么悬空的引脚会读到随机值——电磁干扰让它在不确定区里乱逛。用上拉/下拉电阻可以解决这个问题。
:::

## 电流限制

单片机引脚的设计目的是传递**信号**，而不是提供**动力**。

| 参数 | 典型值 | 含义 |
|------|--------|------|
| 单个引脚最大电流 | ±20mA | 一个引脚最多能输出或吸入 20 毫安 |
| 芯片总电流 | ~150mA | 所有 GPIO 引脚电流的总和上限 |

20mA 够做什么？不够做什么？

| 够用 | 不够用（危险） |
|------|---------------|
| 点亮 1 颗 LED（配限流电阻） | 驱动电机 |
| 驱动逻辑信号到其他芯片 | 驱动大功率 LED |
| 读取按钮状态（几乎不耗电） | 驱动继电器线圈 |
| 驱动三极管的基极 | 驱动蜂鸣器（部分大功率型号） |

::: tip 一个 LED 需要多大电流？
普通红色 LED 在 10mA 时就足够亮了。3.3V 供电，LED 压降约 2V，需要串联一个电阻：

```
R = (3.3V - 2.0V) / 0.01A = 130Ω
```

实际使用 220Ω 或 330Ω（电流更小、更安全，亮度也完全可以接受）。**永远不要省掉这个电阻**——没有限流电阻，LED 会在几秒内烧毁，还可能连带损坏引脚。
:::

## 5V 耐受引脚（FT）

部分 STM32 引脚在数据手册中标注了 **FT（Five-volt Tolerant）**。这意味着这些引脚在设计时做了特殊保护，可以接受 5V 的输入信号。

::: warning 5V 耐受的注意事项
- **只有输入模式才能承受 5V**。即使 FT 引脚，设为推挽输出后输出 3.3V，但你给它外部灌 5V 输入也可能出问题
- **不是所有引脚都是 FT**。必须查数据手册（Datasheet）确认
- **MSPM0G3507 没有 FT 引脚**——所有引脚最高只能承受 3.6V
:::

查数据手册（Pin Definitions 表格）时，找到目标引脚，看 I/O Level 一栏：
- `FT` = 5V 耐受，安全
- `TC` = 标准 3.3V，只能接 3.3V
- 空着或者标其他 = 以 3.3V 为准

## 绝不能直接连接的设备

| 设备 | 为什么不能直接接 | 正确做法 |
|------|----------------|---------|
| 直流电机 | 启动电流可达几百 mA，远超 20mA | 用 L298N/TB6612 等电机驱动模块 |
| 舵机 | 信号可以直连，但**电源**必须外接 | 舵机的 VCC 接电池或外部 5V，GND 与单片机共地，信号线接 GPIO |
| 继电器 | 线圈需要 50~100mA 才能吸合 | 用三极管 + 续流二极管驱动 |
| 大功率 LED（1W+） | 需要 350mA+ 恒流 | 用专用恒流驱动芯片 |
| 5V 传感器 | 输出信号可能是 5V 电平 | 用分压电阻（5V→3.3V）或电平转换模块 |

## 实操守则

1. **接线前，先用万用表测电压**。不确定一根线输出几伏？测一下，只花 5 秒钟，省下一块板子。
2. **永远给 LED 串联限流电阻**。220Ω 是最安全的默认值。
3. **电机和继电器 100% 需要驱动电路**。GPIO 只提供控制信号，不提供动力。
4. **所有模块的 GND 必须连在一起（共地）**。不同电路板之间信号传递的前提是参考电压一致。
5. **先断电，再接线**。带电插拔引脚容易产生瞬间电压尖峰，损伤芯片。
6. **怀疑芯片坏了？先检查这几个**：是否有引脚短路（万用表蜂鸣档）、芯片是否发热（手指轻触）、电源 LED 是否亮起。

## 小结

| 你该记住的 | 具体数值 |
|-----------|---------|
| 工作电压 | 3.3V |
| HIGH 输出 | 约 3.3V |
| LOW 输出 | 约 0V |
| 输入判为 HIGH | > 2.0V |
| 输入判为 LOW | < 0.8V |
| 单引脚最大电流 | ±20mA |
| LED 限流电阻 | 220Ω~330Ω |
| 驱动电机 | 用驱动模块，不直连 |
| 5V 信号 | 仅 FT 引脚可用 |
