# ADC 与 DAC

## 模拟世界 vs 数字世界

单片机内部全是 0 和 1（数字）。但真实世界不是——温度是连续变化的，声音是连续的波形，光强是连续变化的亮度。要让单片机和真实世界对话，就需要两座桥梁：

- **ADC**（模数转换器）：模拟 → 数字（把电压变成数字）
- **DAC**（数模转换器）：数字 → 模拟（把数字变成电压）

```
真实世界                          单片机世界
（模拟）                          （数字）

传感器输出 ────→ ADC ────→ 数字值（0~4095）
  (0~3.3V)       ↑
                模拟→数字

喇叭/电机 ←──── DAC ←──── 数字值（0~4095）
  (0~3.3V)       ↑
                数字→模拟
```

## ADC 的核心参数

| 参数 | 含义 | STM32F103 典型值 | 怎么理解 |
|------|------|-----------------|---------|
| **分辨率** | 能把电压分成多少份 | 12 位 | 2^12 = 4096 份，3.3V / 4096 ≈ 0.8mV 每份 |
| **参考电压** | ADC 量程的上限 | 3.3V | 输入超过 3.3V → 读出来是 4095（饱和了） |
| **采样率** | 一秒钟能测多少次 | 最大 1MHz | 够测音频（44.1kHz），电赛里的传感器最多几十 Hz |
| **通道数** | 能同时/轮询测几路信号 | 最多 16 路 | 一般的 MCU 有 8~16 个 ADC 输入引脚 |

### 分辨率其实很简单

```
12 位 ADC，参考电压 3.3V：

  输入电压            ADC 读数
  ─────────          ────────
  0.0V      ←────→   0
  1.65V     ←────→   2048     （中间值）
  3.3V      ←────→   4095     （满量程）

每 1 个读数单位 ≈ 3.3V / 4096 ≈ 0.8mV
```

搞懂了这个，你就能把 ADC 读数翻译成真实电压：

```c
uint16_t adc_val = HAL_ADC_GetValue(&hadc1);      // 0~4095
float voltage = adc_val * 3.3f / 4096.0f;          // 转为伏特
```

## STM32 ADC 使用要点

```c
// CubeMX 配置好 ADC1 + 通道，然后：
HAL_ADC_Start(&hadc1);
if (HAL_ADC_PollForConversion(&hadc1, 100) == HAL_OK) {
    uint32_t raw = HAL_ADC_GetValue(&hadc1);
}
```

::: warning ADC 采多了会阻塞
上面是**轮询模式**——CPU 等着 ADC 采完。如果你每毫秒采一次还 OK，但连续采几千个点就会拖慢主循环。连续采样应该用 **DMA 模式**（见第五篇 DMA 章节），让 ADC 自己往内存里写数据，CPU 该干吗干吗。
:::

## DAC：反过来

DAC 是 ADC 的逆过程——给一个数字值，输出对应的模拟电压。

电赛里 DAC 的典型应用：
- 输出特定波形（正弦波、三角波）做信号源
- 控制需要模拟电压的设备（一些老式功放、可调电源）
- 生成音频

```c
// STM32 DAC 输出
HAL_DAC_SetValue(&hdac, DAC_CHANNEL_1, DAC_ALIGN_12B_R, 2048);
// 输出 2048/4096 × 3.3V ≈ 1.65V
```

::: tip 电赛里 ADC 远比 DAC 常用
你大部分时间在用 ADC 读传感器——电位器、红外测距、电流检测、电池电压。DAC 用得少得多。如果有 DAC 需求，STM32 自带的 12 位 DAC 基本够用。
:::

## 电赛里 ADC 的典型用途

| 用途 | 传感器 | ADC 读的是什么 |
|------|--------|---------------|
| 电池电压检测 | 电阻分压 | 电压值 → 判断电量 |
| 电流检测 | ACS712 / INA219 | 电压值 → 换算成电流 |
| 灰度巡线 | 灰度传感器 | 多个 ADC 通道 → 灰度值 |
| 电位器 | 旋转电位器 | 转角 → ADC 读数 |
| 红外测距 | GP2Y0A21 | 距离 → 模拟电压 |

## 小结

ADC 是单片机感知模拟世界的唯一通道。12 位分辨率 + 3.3V 参考电压，一个读数对应 0.8mV。读传感器、测电压、检测电流全是用 ADC。DAC 是反向操作，用得少但知道怎么用。连续大量采样记得用 DMA。
