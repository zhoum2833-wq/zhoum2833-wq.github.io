# RAM 与 ROM（Flash）

**ROM** （只读存储器，单片机里主要是 Flash），用于存放提前烧录好的程序代码、固定常量、查表数据；断电后数据不会丢失。正常运行过程中仅支持读取，修改需要专用下载器或特殊擦写指令。

**RAM**（随机存取存储器）相当于是内存（功能等同于电脑内存），程序运行时临时存放全局变量、局部变量、运算中间数据，同时划分出一段区域作为堆栈（栈 Stack）。
函数调用、中断响应、递归运算都会向栈中压入数据；如果递归深度过大、局部数组定义过大，会超出栈分配空间，引发栈溢出，造成程序死机、跑飞，是嵌入式开发中十分常见的内存故障。
