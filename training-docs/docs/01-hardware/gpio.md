# GPIO 与片内外设

单片机的引脚可以配置为输入或输出，这是它和外部世界交互的基础。这种可编程配置的引脚就叫 GPIO（通用输入输出）。

## 输出

输出模式有两种：

**推挽输出**是最常用的模式。它能主动输出高电平或低电平，驱动能力强。点灯、发信号都用它。

**开漏输出**只能主动拉低，不能主动拉高。要输出高电平，必须外接上拉电阻。I2C 总线和多设备共享信号线时用它。

## 输入

输入模式用来读取外部信号——按键按下没有、传感器输出是高是低。如果引脚悬空（什么都没接），电压不定，读到的是什么完全看运气。

解决方法：启用内部上拉或下拉电阻，让引脚在悬空时有一个确定的默认电平。

## 片内外设

同一根引脚可以复用为外设的专用接口。以下是常用外设：

| 外设 | 作用 |
|------|------|
| UART | 两个芯片之间串行通信 |
| I2C | 两根线挂多个传感器 |
| SPI | 四线高速通信 |
| ADC | 把电压转成数字量 |
| PWM | 通过占空比控制电机转速、LED 亮度 |
| Timer | 精确计时、测量脉冲宽度 |
