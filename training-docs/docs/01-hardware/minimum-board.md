# 最小系统

单片机芯片不能仅靠供电就跑起来。它需要一些最基本的外围电路才能复位、启振并工作。这些加起来就是最小系统。

## 四个必要条件

| 组件 | 作用 |
|------|------|
| 电源 | 提供稳定的 3.3V |
| 晶振 | 提供时钟信号 |
| 复位电路 | 上电时让 CPU 从第一条指令开始执行 |
| 下载接口 | 把程序烧进去 |

**少一个，芯片就跑不起来。**

## 为什么用最小系统板

裸芯片引脚间距只有 0.5mm，手工焊接困难，也插不进面包板。最小系统板把芯片和最小外围电路做在一块 PCB 上，用 2.54mm 间距的排针把所有引脚引出来——正好兼容面包板和杜邦线。

## 板上有什么

- **单片机芯片**：正中央的大芯片
- **晶振**：银色金属壳元件，标着 8.000（8MHz）。经内部 PLL 倍频后得到芯片的主频。另一颗 32.768KHz 晶振专为实时时钟提供基准
- **复位按钮**：按下后芯片从头执行程序
- **LDO 稳压器**：把 USB 的 5V 降到 3.3V
- **BOOT0 跳线**：正常运行时插在 0，需要串口下载时拨到 1，下载完拨回 0
- **排针**：间距 2.54mm，通过杜邦线连接外设
