# 定时器与 PWM

## 定时器是什么

定时器（Timer）本质就是**一个自动数数的计数器**。每个时钟周期，计数器加 1。加到某个值——触发事件（中断、或改变引脚电平）。

```
  时钟脉冲: ┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐  (72MHz → 每个脉冲 13.9ns)
  计数器:   0→1→2→3→...→(ARR)→0→1→...

  预分频器（PSC）: 先把 72MHz 降速，比如 ÷72 → 1MHz
  自动重载值（ARR）: 计数器从 0 数到 ARR，然后归零
```

**定时器溢出频率 = 定时器时钟 / (PSC + 1) / (ARR + 1)**

举例：72MHz 时钟，PSC=71（÷72），ARR=999 → 1MHz / 1000 = **1kHz** 溢出频率。

## PWM：定时器最常用的功能

PWM（Pulse Width Modulation，脉冲宽度调制）用定时器的比较输出功能产生方波。关键参数：

```
周期（Period）决定频率
  |←─────── 一个周期 ──────→|
  ┌──────────┐              ┌──
  │          │              │
  │   高电平  │   低电平      │
  │          │              │
  └──────────┘              └──
  |← 脉宽 →|

占空比 = 脉宽 / 周期 × 100%
```

- **占空比 50%**：一半高一半低 → 电机半速
- **占空比 75%**：¾ 高 ¼ 低 → 电机 75% 速度
- **占空比 100%**：全高 → 电机全速

## 三个典型应用

### 1. 电机调速

```c
// TIM2 CH1 输出 PWM，连到 TB6612 的 PWMA 引脚
// 占空比 = CCR / ARR
TIM_HandleTypeDef htim2;

// CubeMX 配置：TIM2, CH1, PWM Mode 1, 20kHz
HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1);
__HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 500);  // 500/1000 = 50% 占空比
```

::: tip PWM 频率选多少
电机驱动推荐 **10kHz~20kHz**。低于 1kHz → 电机发出刺耳啸叫（人耳能听到）。高于 20kHz → 开关损耗增大。舵机固定用 **50Hz**。
:::

### 2. 舵机控制

舵机控制信号是 50Hz PWM，靠**脉冲宽度**而不是占空比表示角度：

```
  周期 20ms（50Hz）
  |<────────────────────>|

  ┌─┐  0.5ms  = 0°
  └─┘
  ┌───┐  1.5ms = 90°
  └───┘
  ┌─────┐  2.5ms = 180°
  └─────┘
```

配置：PSC=71, ARR=19999 → 72MHz / 72 / 20000 = **50Hz**
CCR = 500 → 0.5ms → 0°，CCR = 1500 → 1.5ms → 90°，CCR = 2500 → 2.5ms → 180°

### 3. LED 呼吸灯

LED 的亮度和 PWM 占空比成比例——利用人眼的视觉暂留效果。

```c
// 亮度从 0 到最大再回到 0（呼吸效果）
for (int i = 0; i <= 1000; i += 10) {
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, i);
    HAL_Delay(10);
}
```

## 输入捕获：定时器也能"听"

除了输出 PWM，定时器还能**测量**外部信号的频率和脉宽。比如测编码器脉冲的频率算速度，或者测超声波回波的时间算距离。

原理：捕获引脚上电平变化时的计数值。两次上升沿之间的计数值差 = 信号周期。

## 小结

定时器 = 精准的数数器。配好 PSC（预分频）+ ARR（自动重载）→ 得到你想要的频率。PWM = 定时器比较输出 → 控制了电机、舵机、LED 的"力道"。电机 PWM 用 10~20kHz，舵机用 50Hz。
