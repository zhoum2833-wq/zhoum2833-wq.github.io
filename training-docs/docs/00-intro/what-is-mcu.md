# 什么是单片机

## 一句话定义

**单片机（MCU，Microcontroller Unit）是一颗芯片上的完整计算机。**

它把 CPU（中央处理器）、内存（RAM）、硬盘（Flash）、输入输出接口（GPIO）全部集成在一块指甲盖大小的硅片上。插上电，烧录程序，它就能独立工作。

## 类比：笔记本电脑 vs 单片机

想象你的笔记本电脑：有英特尔处理器负责计算，有 16GB 内存运行程序，有 512GB 固态硬盘存放文件，还有 USB 口、耳机孔这些外接接口。

单片机就是把这套结构缩小一千万倍，塞进一颗芯片里：

| 组件 | 笔记本电脑 | 单片机（以 STM32F103 为例） |
|------|-----------|---------------------------|
| 大脑（CPU） | Intel Core i7 @ 3GHz | ARM Cortex-M3 @ 72MHz |
| 临时记忆（RAM） | 16GB DDR4 | 64KB SRAM |
| 永久存储（ROM/Flash） | 512GB SSD | 512KB Flash |
| 对外接口 | USB、HDMI、耳机孔 | GPIO 引脚、UART、I2C、SPI |
| 价格 | 5000+ 元 | 不到 20 元 |

单片机的各项参数看起来比电脑小了成千上万倍，但它也便宜了成千上万倍——而且对于控制一台微波炉、一台洗衣机来说，这些资源绰绰有余。

## 单片机做什么？

所有单片机程序，归根结底只有三步循环：

```
感知（Read Sensors）→ 决策（Process Data）→ 执行（Control Actuators）→ 回到感知
```

- **感知**：读取传感器数据。比如温度传感器传来的电压值、按钮是否被按下、红外探头检测到的障碍物距离。
- **决策**：处理数据，做出判断。比如"当前温度低于设定值，需要加热"。
- **执行**：控制输出设备。比如打开加热器、点亮 LED、让电机转动。

这个过程每秒循环成千上万次，所以它能实时响应外界变化。

## 为什么叫"单片机"？

把"单片微型计算机"拆开看：

- **单片**：所有功能集成在**一颗**芯片上。不需要外接内存条，不需要外接硬盘，通电就能跑。
- **微型**：体积小、功耗低、性能适中。不是用来跑 Windows 的，是用来控制设备的。
- **计算机**：它有 CPU，能执行程序，能做运算和逻辑判断。

英文叫 Single-Chip Microcomputer，直译就是"单片微型计算机"。

## 你身边到处都是单片机

单片机的产量比电脑 CPU + 手机 CPU 加起来还要多几十倍。你家里至少有十几颗单片机正在工作：

- **微波炉**：旋钮和数字显示屏背后是一颗 MCU，它读取你的按键，控制加热时间，到时间"叮"一声。
- **洗衣机**：MCU 控制进水阀、排水泵、滚筒电机，按预设程序完成洗涤→漂洗→脱水的全过程。
- **空调遥控器**：一颗 4 位或 8 位 MCU，成本几毛钱，负责读取按键、驱动红外发射管。
- **无人机飞行控制器**：一颗 STM32 系列 MCU，每秒读取陀螺仪和加速度计数据数百次，实时调整四个电机的转速保持平衡。
- **电子竞赛智能小车**：就是我们接下来要做的。

## 一块典型的单片机开发板长什么样？

想象你面前有这样一块板子：

一块绿色的长方形电路板，大约 5cm x 5cm。正中央是一颗黑色的正方形芯片——这就是单片机本体，表面印着"STM32F103C8T6"的型号。芯片四周伸出四排银色的金属引脚，像蜈蚣的腿，总共 48 根。旁边有一颗小指头指甲盖大小的银色椭圆元件——那是晶振，给单片机提供稳定的心跳信号。角落里有几个 LED 指示灯和一个小按钮。板子上还有一个 Micro-USB 口，插上数据线就能连接电脑，下载程序、供电两不误。

## 我们要用它做什么？

在电子竞赛中，我们用单片机来**搭建一辆智能小车**。这辆小车需要：

- 沿着地面上的黑色线条自动行驶（**循迹**）
- 穿过迷宫找到出口（**迷宫导航**）
- 识别赛道上各种标记和路障（**环境感知**）
- 完成赛题要求的各种动作和任务（**任务执行**）

从点亮一个 LED 灯开始，你会逐步学会如何让一颗芯片"活"起来，掌控电机、传感器、显示器……最终做出一辆能独立思考的小车。
