# 单片机的发展演化

## 从 8 位到 32 位：一部浓缩史

单片机的发展史就是一部"更大、更快、更强"的进化史。我们用四个时代来概括：

```
1980s              2000s              2010s              2020s
  |                  |                  |                  |
8051               AVR              STM32              MSPM0
8位                 8位              STM32F103          ESP32
@ 12MHz             @ 16MHz           @ 72MHz            @ 240MHz
                    (Arduino 时代)    (竞赛主力)         (现代 MCU)
```

## 第一代：8051 时代（1980s~1990s）

Intel 在 1980 年推出的 8051 是单片机的"活化石"，至今仍在一些简单产品中使用。

| 特性 | 参数 |
|------|------|
| CPU 位数 | 8 位（一次处理 8 个二进制位） |
| 主频 | 通常 12MHz |
| 内存 | 128 字节 RAM，4KB ROM |
| 编程方式 | 并行烧录器，需要紫外线擦除 |
| 典型应用 | 早期家电、工业控制器 |

它的设计简洁可靠，但性能极其有限：128 字节 RAM 甚至不够存放一个较长的字符串。今天你可以在网上花 3 块钱买到一块 8051 最小系统板，它依然能工作——但也仅此而已。

## 第二代：AVR / Arduino 时代（2000s）

Atmel 公司（后被 Microchip 收购）推出的 AVR 系列单片机带来了重大改进。尤其是 Arduino 平台的出现，让单片机编程从专业工程师走向了普通爱好者。

| 特性 | 参数 |
|------|------|
| CPU 位数 | 8 位 |
| 主频 | 16MHz |
| 内存 | 2KB SRAM，32KB Flash |
| 编程方式 | USB 直接烧录，IDE 一键下载 |
| 典型应用 | Arduino Uno、3D 打印机控制板 |

Arduino 的最大贡献不是硬件本身，而是**降低了入门门槛**：不需要学汇编，不需要懂寄存器，用 C++ 写几行 `digitalWrite(13, HIGH)` 就能点亮 LED。这个生态至今影响着全世界成千上万的创客。

## 第三代：STM32 / ARM Cortex 时代（2010s）

STMicroelectronics（意法半导体）推出的 STM32 系列是竞赛领域的事实标准。它采用了 ARM 公司设计的 Cortex-M3 内核，性能相比 AVR 提升了不止一个数量级。

| 特性 | 参数 |
|------|------|
| CPU 位数 | 32 位 |
| 主频 | 72MHz（STM32F103） |
| 内存 | 20KB~64KB SRAM，64KB~512KB Flash |
| 编程方式 | SWD 调试器、USB DFU |
| 典型应用 | 电子竞赛、无人机飞控、工业变频器 |

为什么 STM32 能成为竞赛首选？

- **价格低**：一块 STM32F103C8T6 最小系统板只要 10~20 元
- **性能够**：72MHz 主频 + 32 位 Cortex-M3，处理循迹、PID 控制绰绰有余
- **资料多**：中文教程、示例代码、视频讲解铺天盖地
- **生态好**：HAL 库、CubeMX 配置工具、Keil/IAR/GCC 多平台支持

## 第四代：现代 MCU（2020s）

现在的单片机市场百花齐放：

| 芯片 | CPU 核 | 主频 | 特色 |
|------|--------|------|------|
| MSPM0G3507 | Cortex-M0+ | 32MHz | TI 出品，高性价比，模拟外设强 |
| ESP32 | 双核 Xtensa | 240MHz | 内置 WiFi + 蓝牙 |
| RP2040 | 双核 Cortex-M0+ | 133MHz | Raspberry Pi 出品，PIO 灵活接口 |
| STM32G4 | Cortex-M4 | 170MHz | 带 FPU 和 DSP，适合电机控制 |

## 发展趋势：四个方向

1. **位数增长**：4 位 → 8 位 → 16 位 → 32 位（64 位在 MCU 领域极少见，暂时不需要）
2. **主频飙升**：KHz → MHz → 数百 MHz → 接近 GHz
3. **功能集成**：单核 → 多核；无连接 → WiFi/BLE/USB/以太网内置
4. **功耗降低**：mA 级 → μA 级休眠电流，纽扣电池供电几年不是问题

## 我们使用什么？

本次培训使用两种芯片：

::: tip 我们使用的两款 MCU

- **STM32F103**：Cortex-M3 @ 72MHz，电子竞赛的经典方案，成熟可靠
- **MSPM0G3507**：Cortex-M0+ @ 32MHz，TI 的新一代入门芯片，性价比突出

两者都是 32 位 ARM 架构，编程方式相近，学会一个就能触类旁通。
:::

## 为什么竞赛不用更高级的芯片？

你可能会问：240MHz 双核的 ESP32 也只卖 20 元，为什么竞赛不用？

三个原因：
1. **够用就好**：循迹小车不需要 WiFi，72MHz 的单核 MCU 完全胜任
2. **简单可靠**：芯片越复杂，配置越多、坑越多。竞赛中稳定性是第一位的
3. **规则限制**：部分竞赛对主控芯片型号有要求，STM32F103 是被广泛认可的型号

当你掌握了基础之后，自然可以根据需求选择更强的芯片。
