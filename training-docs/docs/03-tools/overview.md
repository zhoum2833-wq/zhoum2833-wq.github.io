# 开发工具全景

电赛开发的工具链分三块：

## CubeMX：生成初始化代码

用图形界面配置时钟、引脚和外设参数，一键生成 Keil 工程和初始化代码。

## Keil5：编译和烧录

Keil5 是 KEIL 公司（现属 ARM）的 IDE。免费社区版支持 32KB 以内的代码，竞赛够用。

直接打开 CubeMX 生成的 `.uvprojx` 工程文件，点编译、点下载就完了。不需要自己搭 GCC 工具链——电赛抢时间，能跑就行。

## VS Code：写代码 + AI 辅助

VS Code 是微软的免费代码编辑器。用它来编辑 C 代码、浏览工程文件、和 AI（Claude Code）结对编程。

为什么不直接用 Keil5 写代码？Keil5 的编辑器功能有限，没有 AI 插件，界面也不如 VS Code 方便。

## 实际工作流

```
CubeMX 配引脚/时钟 → 生成 Keil 工程
                           ↓
VS Code 打开工程文件夹，写 C 代码，AI 辅助
                           ↓
Keil5 打开 .uvprojx，编译、下载、调试
```

三个工具各司其职，不折腾开源工具链。
