# AI 辅助嵌入式开发

## 怎么用好 AI？

**人负责架构和逻辑，AI 负责落地编码**

### 写清楚 CLAUDE.md

CLAUDE.md 是给 AI 看的"项目说明书"。4WD_Car 的 CLAUDE.md 包含：
- 项目架构（三层结构、调用规则）
- 引脚分配表
- 编码规范（命名风格、文件头格式）
- 初始化顺序
- 注意事项（已踩过的坑）

有了这个文件，AI 就像一个已经接手项目两周的队友，你不需要每次都从头解释项目背景。

### 给 AI 看参考代码

直接说"帮我写一个电机驱动"太模糊了。给它一个已有的 BSP 模块做参考：
> "参考 bsp_uart.c 的风格和模板，帮我写 bsp_xxx.c"

AI 会模仿你的编码风格（缩进、命名、注释格式），不会造出风格迥异的代码。
