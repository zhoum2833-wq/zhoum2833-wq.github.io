# 隔离的作用与目的

## 为什么要隔离？

在嵌入式开发中，你经常用 USB 线把开发板连到电脑上调程序。大部分时候一切正常——直到你开始调试**电机驱动**。

::: danger 这不是吓唬你
电机在关断瞬间会产生**反向电动势**（Back-EMF），电压可以瞬间飙到几十甚至上百伏。如果这个高压通过地线、信号线或电源线倒灌回电脑 USB 口，可能直接把 USB 控制器甚至是主板烧掉。修一次几百到几千块不等。
:::

打比方：隔离就像医院的**隔离病房**。传染病人（电机噪声/高压）和健康人（电脑）之间有一道物理屏障，病菌（高压）过不去，但医生还能透过窗户（隔离信号）看到病人情况。

## 电气隔离 vs 非隔离

电气隔离（Galvanic Isolation）是指在两个电路之间**没有任何直接的导电路径**，信号和电源都通过磁场（变压器）、电场（电容耦合）或光（光电耦合器）传递。

```c
// 非隔离连接（危险！）
电脑 USB ─────── 导线 ─────── STM32 开发板
                                  │
                                  ├── GPIO ── 电机驱动 ── 电机
                                  │
        // 电机反压通过 GPIO → MCU → USB → 电脑！
        // 电流路径存在 → 高压找到回家的路

// 隔离连接（安全）
电脑 USB ── 隔离芯片 ─┊┊┊┊ 隔离屏障 ┊┊┊┊── STM32 开发板
                       （磁场耦合）            │
                                         电机驱动 ── 电机
        // 屏障两侧没有导线连接 → 高压无路可走
```

## USB 隔离器

竞赛调试时**最重要的保护设备**，没有之一。

一个 USB 隔离器内部包含一块**隔离芯片**（常见型号 **ADuM3160**），通过芯片内部的微型变压器传递 USB 信号，而电源通过隔离型 DC-DC 模块独立提供。物理上，输入和输出之间没有任何铜导线连接。

市面上成品 USB 隔离器几十元到上百元不等。如果你经常调试电机驱动板，这个投资极为划算。

**什么时候必须用 USB 隔离器：**

- 调试任何涉及电机的电路板
- 调试高压电路（例如电池串联到 24V+）
- 电路板从外部大功率电源取电
- 调试自己画的 PCB（第一次通电，可能有短路或设计错误）

::: tip 经验
养成习惯：调试新板子时，第一件事不是插 USB，而是先用万用表量一下 3.3V 和 GND 是否短路。确认无误后，再用 USB 隔离器连接。等板子完全调通，确认无高压风险后，才考虑直连。
:::

## 其他隔离方式

### 光电耦合器（Optocoupler）

**原理**：电信号 → LED 发光 → 光敏三极管接收 → 恢复成电信号。输入输出之间只有光路，没有电路。

**典型型号**：PC817（低速）、6N137（高速）。用于数字信号隔离，如隔离 UART、隔离 GPIO。

**优点**：便宜（几毛钱）、单向、耐压高（数千伏）。

**缺点**：速度慢（普通光耦只能到几十 kHz）、需要限流电阻。

### 隔离型 DC-DC 模块

普通 DC-DC 输入输出的地在内部是通的（非隔离）。隔离型 DC-DC（如 B0505S-1W）输入和输出的地完全分离，各自独立。

**用途**：为隔离后的电路单独供电。例如隔离型 CAN 总线、隔离型 RS-485 电路中，总线侧的芯片需要用隔离电源供电。

### 继电器

继电器也是隔离元件——控制线圈和触点之间没有电气连接。用低压控制高压（如 3.3V 控制 220V 交流）时，继电器提供物理隔离。

## 隔离的应用场景总结

| 场景 | 隔离方式 | 隔离等级 |
|------|----------|----------|
| 调试电机板子 | USB 隔离器 (ADuM3160) | 几千伏 |
| 数字信号传递 | 光耦 (PC817/6N137) | 几千伏 |
| 隔离电源 | 隔离 DC-DC (B0505S) | 1000~3000V |
| 高压通断 | 继电器 | 几千伏 |
| CAN/RS-485 总线 | 隔离收发器 | 几千伏 |
| 高压测量 | 隔离运放 / 霍尔传感器 | 几千伏 |

---

::: warning 竞赛特别提醒
比赛现场电脑是**你自己的**，烧了没人赔。所以：

1. 所有电机驱动相关调试必须经过 USB 隔离器
2. 备一个备用 USB 隔离器（东西不贵，关键时刻救命）
3. MCU 和电机之间尽量用光耦隔离关键信号
4. 电机独立供电，地和 MCU 地必要时才单点连接
:::

## 小结

隔离的核心目的：保护你的电脑。电机转动时会产生反向电动势和电磁干扰，通过 USB 线传回电脑可能烧毁 USB 口甚至主板。隔离方案的选择：低速信号用光耦，高速信号用磁耦或隔离芯片，预算紧张时至少给调试口加一个 USB 隔离器。
