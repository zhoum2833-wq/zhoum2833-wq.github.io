# TB6612FNG — 两路电机驱动

## 它是什么

TB6612FNG 是一颗**直流电机驱动芯片**。它像一个"电机翻译官"——MCU 给它低电压的逻辑信号（3.3V，几乎不费力），它输出大电流（最大 1.2A 每路）来驱动电机。

把它想象成音响的功放：手机输出微弱的音频信号，功放把它放大到能驱动喇叭。TB6612 把 MCU 的弱信号"放大"成能驱动电机的强电流。

## 它能做什么

- 同时驱动 **2 个** 直流电机（小车左轮 + 右轮）
- 每个电机可以 **正转、反转、刹车、滑行**
- 用 **PWM（脉冲宽度调制）** 控制速度
- 工作电压：电机供电 2.5V-13.5V，逻辑供电 2.7V-5.5V

## 关键引脚

| 引脚名 | 功能 | 接哪里 |
|--------|------|--------|
| VM | 电机电源 | 电池正极（如 7.4V 锂电池） |
| VCC | 芯片逻辑电源 | 3.3V 或 5V |
| GND | 地 | 电池负极 + MCU 地 |
| PWMA | A 路电机速度 | MCU 的 PWM 输出引脚 |
| AIN1, AIN2 | A 路电机方向 | MCU 的两个普通 GPIO |
| PWMB | B 路电机速度 | MCU 的 PWM 输出引脚 |
| BIN1, BIN2 | B 路电机方向 | MCU 的两个普通 GPIO |
| STBY | 使能（总开关） | MCU 的一个 GPIO，拉高 = 工作 |
| AO1, AO2 | A 路输出 | 接 A 电机的两根线 |
| BO1, BO2 | B 路输出 | 接 B 电机的两根线 |

## 真值表：怎么控制电机

以 A 路电机为例（B 路完全一样）：

| AIN1 | AIN2 | PWMA | 电机状态 | 什么时候用 |
|------|------|------|---------|-----------|
| 1 | 0 | 0-100% | **正转**（前进） | 让车前进 |
| 0 | 1 | 0-100% | **反转**（后退） | 让车后退 |
| 1 | 1 | 任意 | **刹车**（急停） | 需要立刻停下 |
| 0 | 0 | 任意 | **滑行**（自由停） | 让车自然减速 |
| 0 | 0 | 0 | **停止** | 省电待机 |

这里 1 = 高电平（3.3V），0 = 低电平（0V），PWM 的百分比 = 速度（0% 不动，100% 全速）。

::: tip 刹车 vs 滑行
- **刹车（1,1）：** 电机两个端子被短接到地，电机像被"夹住"一样立刻停下。  
- **滑行（0,0）：** 电机两个端子断开，电机自由转动，车靠摩擦力慢慢停。
:::

## PWM 频率设置

PWM 频率建议设为 **10kHz**（即周期 0.1ms）。原因：

- 低于 10kHz → 电机会发出"滋滋"的高频噪音（人耳能听到 20Hz-20kHz）
- 高于 100kHz → 驱动芯片的开关损耗增大，发热严重
- **10kHz-20kHz 是最佳区间**：人耳听不到，芯片也不会过热

```c
// 以 MSPM0 为例，设置 PWM 频率为 10kHz
// 假设定时器时钟为 48MHz
// PWM 频率 = 定时器时钟 / (预分频 × 自动重装载值)
// 10kHz = 48MHz / (48 × 100) → 预分频=48, 重装载=100
```

## 死区时间（Dead Time）

::: danger 重要安全规则
**在电机正转时，绝对不要直接切换到反转！**

必须先刹车或滑行一小段时间（约 1-5ms），再切换到反向。否则可能出现"上下桥臂同时导通"（shoot-through），瞬间大电流烧毁芯片。
:::

```c
// 错误写法（危险！）
void motor_change_direction_bad() {
    motor_forward(80);   // 正在前进
    motor_reverse(80);   // 直接反转 → 可能烧芯片！
}

// 正确写法（安全）
void motor_change_direction_good() {
    motor_forward(80);   // 正在前进
    motor_brake();       // 先刹车
    delay_ms(5);         // 等 5ms（死区时间）
    motor_reverse(80);   // 再反转 → 安全！
}
```

## 代码示例

```c
// ==================== tb6612.h ====================
#ifndef TB6612_H
#define TB6612_H

#include <stdint.h>

void motor_init(void);
void motor_a_forward(uint8_t speed);   // speed: 0-100
void motor_a_reverse(uint8_t speed);
void motor_a_brake(void);
void motor_a_coast(void);
void motor_b_forward(uint8_t speed);
void motor_b_reverse(uint8_t speed);
void motor_b_brake(void);
void motor_b_coast(void);

// 便捷函数：同时控制两路
void car_forward(uint8_t speed);
void car_backward(uint8_t speed);
void car_turn_left(uint8_t speed);     // 左轮慢、右轮快
void car_turn_right(uint8_t speed);    // 左轮快、右轮慢
void car_stop(void);

#endif

// ==================== tb6612.c ====================
#include "tb6612.h"
#include "ti_msp_dl_config.h"   // MSPM0 的配置（你的 MCU 可能不同）

// GPIO 引脚定义（根据你的接线修改）
#define AIN1_PORT  GPIOA
#define AIN1_PIN   DL_GPIO_PIN_0
#define AIN2_PORT  GPIOA
#define AIN2_PIN   DL_GPIO_PIN_1
#define BIN1_PORT  GPIOA
#define BIN1_PIN   DL_GPIO_PIN_2
#define BIN2_PORT  GPIOA
#define BIN2_PIN   DL_GPIO_PIN_3
#define STBY_PORT  GPIOA
#define STBY_PIN   DL_GPIO_PIN_4

// 初始化电机驱动
void motor_init(void) {
    // 假设 GPIO 和 PWM 已在配置工具中初始化
    // 只需设置 STBY 为高电平（使能）
    DL_GPIO_setPins(STBY_PORT, STBY_PIN);
}

// A 路电机：前进
void motor_a_forward(uint8_t speed) {
    DL_GPIO_setPins(AIN1_PORT, AIN1_PIN);      // AIN1 = 1
    DL_GPIO_clearPins(AIN2_PORT, AIN2_PIN);    // AIN2 = 0
    set_pwm_a(speed);  // 设置 PWMA 占空比 = speed%
}

// A 路电机：后退
void motor_a_reverse(uint8_t speed) {
    DL_GPIO_clearPins(AIN1_PORT, AIN1_PIN);    // AIN1 = 0
    DL_GPIO_setPins(AIN2_PORT, AIN2_PIN);      // AIN2 = 1
    set_pwm_a(speed);
}

// A 路电机：刹车
void motor_a_brake(void) {
    DL_GPIO_setPins(AIN1_PORT, AIN1_PIN);      // AIN1 = 1
    DL_GPIO_setPins(AIN2_PORT, AIN2_PIN);      // AIN2 = 1
    set_pwm_a(100);  // 刹车时占空比 100%
}

// A 路电机：滑行
void motor_a_coast(void) {
    DL_GPIO_clearPins(AIN1_PORT, AIN1_PIN);    // AIN1 = 0
    DL_GPIO_clearPins(AIN2_PORT, AIN2_PIN);    // AIN2 = 0
    set_pwm_a(0);
}

// 小车前进（两轮同速）
void car_forward(uint8_t speed) {
    motor_a_forward(speed);
    motor_b_forward(speed);
}

// 小车原地左转（左轮后退、右轮前进）
void car_turn_left(uint8_t speed) {
    motor_a_reverse(speed);
    motor_b_forward(speed);
}

// 小车停止
void car_stop(void) {
    motor_a_coast();
    motor_b_coast();
}
```

::: warning 代码适用说明
以上代码使用 MSPM0 的 `ti_msp_dl_config.h` 和 DriverLib API。如果你使用 STM32、Arduino 或其他平台，GPIO 操作函数名不同，但**控制逻辑完全相同**——核心就是"设置方向引脚 + 设置 PWM 占空比"。
:::

## 接线检查清单

上电前检查以下项目：

- [ ] VM 接电池正极，GND 接电池负极（千万别把 VM 和 VCC 搞混！）
- [ ] VCC 接 3.3V 或 5V（看你的 MCU 逻辑电平）
- [ ] STBY 必须拉高，否则电机完全不转
- [ ] 电机引线拧紧，不要有裸线碰在一起（短路）
- [ ] 芯片表面加散热片（可选，但如果长时间大电流运行建议加）

---

## 小结

TB6612 的用法归结为一张真值表：AIN1/AIN2 控制方向，PWMA 控制速度，STBY 控制使能。接好三个电容、控制信号别超 100kHz，这就是竞赛小车最可靠的电机驱动方案。
