# 什么是调试器

常见的调试器有DAP-Link ，ST-Link 与 J-Link等

## 调试

我们知道嵌入式最重要的能力是调试。调试器有**在线调试**的功能：

- 单步执行代码（一条一条跑）
- 设置断点（运行到某一行代码自动停下）
- 查看变量的实时值（不使用 printf）
- 查看寄存器和内存

## 烧录

虽然说调试器叫调试器，但是其还要一个很重要的功能叫烧录
一般来说，烧录功能都是由调试器承担的

## SWD 协议
