# USB 转串口芯片

## 为什么需要"翻译官"？

你的电脑和单片机（MCU，也叫微控制器，就是你板子上那个小黑方块）说不同的"语言"：

- **电脑的 USB 口**：说 USB 协议（Universal Serial Bus，通用串行总线）。这是一种复杂的高速协议，你的 U 盘、鼠标、键盘都靠它和电脑通信。
- **单片机的 UART 口**：说 UART 协议（Universal Asynchronous Receiver/Transmitter，通用异步收发器）。这是一种简单的低速协议，只有两根线 — 一根发送（TX）、一根接收（RX）。

它们互相听不懂对方在说什么，就像一个人说中文、另一个人说法语。这时候需要一个翻译官 —— **USB 转串口芯片**。

## 翻译官的工作原理

USB 转串口芯片一头接电脑的 USB，一头接单片机的 UART 引脚（TX/RX），在中间做双向翻译：

```
电脑 USB 口  →  USB 数据线  →  USB转串口芯片  →  TX/RX 引脚  →  单片机 UART 口
   (USB协议)                  (CH340/CP2102等)    (UART协议)
```

具体的翻译过程：

1. **电脑发数据给单片机**：电脑通过 USB 发来一包 USB 格式数据 → 芯片把 USB 包拆开，取出里面的真实数据 → 通过 TX 引脚，用 UART 格式一位一位发出去 → 单片机的 RX 引脚收到。
2. **单片机发数据给电脑**：单片机通过 TX 引脚发出 UART 格式数据 → 芯片收到后，把它打包成 USB 格式 → 通过 USB 数据线发给电脑。

这个过程对你是**完全透明的** —— 你写代码时只需要操作串口（Serial Port），不需要关心 USB 协议的任何细节。翻译官帮你搞定了一切。

## 常见的三种翻译官芯片

| 芯片型号 | 厂家 | 价格 | 可靠性 | 特点 |
|---------|------|------|--------|------|
| **CH340** | 南京沁恒（WCH） | 约1元 | 良好 | 国内最常见，性价比最高 |
| **CP2102** | Silicon Labs（美国） | 约5-8元 | 很好 | 稳定，驱动兼容性好 |
| **FT232** | FTDI（英国） | 约15-20元 | 极好 | 工业级，驱动最完善 |

::: tip 你的板子上大概率是 CH340
国内绝大多数开发板（包括正点原子、野火、普中等等）都使用 CH340 系列芯片。它便宜、够用、驱动也好找。如果不是特殊需求，CH340 完全够用。
:::

## 怎么认出你的板子上用的是哪种？

拿出你的开发板，找一块**不是主 MCU 芯片**的小长方形黑色芯片。主 MCU 通常是最大的那块，而 USB 转串口芯片一般在 USB 口旁边，体积小一些。

直接用手机放大拍照，看芯片表面的**丝印（激光刻的字）**：

- **CH340**：芯片上会刻着 `CH340G`、`CH340C`、`CH340N` 或 `CH340K` 字样。`G` 版本需要外接晶振（旁边会有一个银色小铁壳），`C/N/K` 版本内置晶振，外围更简洁。
- **CP2102**：芯片上刻着 `CP2102`，通常是 QFN 封装（正方形，引脚藏在芯片底部，不容易看到）。
- **FT232**：芯片上刻着 `FT232RL` 或 `FT232RQ`，通常比前两种稍大一点。

::: warning 找不到 USB 转串口芯片？
有些板子（如 STM32F103C8T6 最小系统板）的 MCU 本身带 USB 功能，可能直接用 MCU 的 USB 口通信，不在板上放额外的转串口芯片。此外，ST-Link、J-Link、DAP-Link 等调试器本身也集成了 USB 转串口功能，你要看的是调试器上的芯片。
:::

## 芯片需要驱动

CH340 插上电脑不会立刻工作 —— Windows 需要安装对应的**驱动程序**。驱动就是告诉操作系统"这个 USB 设备是什么、怎么跟它说话"的软件。

- **CH340 驱动**：去南京沁恒官网搜索 CH340 驱动下载。安装后在设备管理器里会看到 `USB-SERIAL CH340 (COMx)`。
- **CP2102 驱动**：Silicon Labs 官网下载 CP210x 驱动。
- **FT232 驱动**：FTDI 官网下载，Windows 10/11 通常能自动识别。

装好驱动之后，你就可以在电脑上看到一个 **COM 口**（串行通信端口），例如 `COM3` 或 `COM4`。下一篇文章会详细讲解这个 COM 口是哪里来的。

::: danger 不要在驱动没装好的情况下使用
驱动没装好，串口软件打开 COM 口时会报错或找不到端口。如果你插上板子但串口软件里没有新的 COM 口出现，先检查设备管理器里有没有带黄色感叹号的未知设备。
:::

## 小结

USB 转串口芯片把复杂的 USB 协议翻译成简单的 UART，让你可以在电脑上像用老式串口一样和单片机通信。国内最常见的 CH340 便宜可靠，够你用到比赛结束。
